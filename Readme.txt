How to run the Vehicle Parking Management System Project using PHP and MySQL

1.Extract the file and copy vpms folder

2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html)

3.Open PHPMyAdmin (http://localhost/phpmyadmin)

4.Create a database with the name vpmsdb

5.Import vpmsdb.sql file(given inside the zip package in SQL file folder)

6.Run the script http://localhost/vpms
